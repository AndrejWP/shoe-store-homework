# Практические работы 7 и 8

React-проект с компонентом `ProfileCard` и загрузкой пользователей из API.

## Запуск

```bash
npm install
npm run dev
```

После запуска открыть адрес, который покажет Vite.

## Что реализовано

- компонент `ProfileCard`;
- компонент `UsersList`;
- компонент `UserItem`;
- запрос к `https://jsonplaceholder.typicode.com/users`;
- состояния загрузки и ошибки;
- вывод списка через `map()`.
