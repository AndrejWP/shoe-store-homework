# 👟 BLACK STORE — Интернет-магазин обуви

Учебный проект по дисциплине «Конструирование и сопровождение программ» (КСП).

## О проекте

BLACK STORE — это фронтенд интернет-магазина обуви с каталогом товаров, фильтрацией по категориям и корзиной покупок. Корзина сохраняется в `LocalStorage`, поэтому товары не пропадут после перезагрузки страницы.

## Функционал

- Каталог товаров с карточками (название, фото, описание, цена)
- Фильтрация по категориям: кроссовки, ботинки, классическая обувь
- Корзина: добавление, удаление товаров, подсчёт итоговой суммы
- Сохранение корзины в `LocalStorage`
- Оплата и очистка корзины

## Стек

- HTML5
- CSS3
- JavaScript (Vanilla)

## Структура проекта

```
mysite/
├── index.html        # Главная страница
├── catalog.html      # Каталог товаров + корзина
├── item.html         # Страница товара
├── script.js         # Логика корзины и LocalStorage
├── style.css         # Стили
└── img/              # Изображения товаров
```

## Запуск

Открыть `index.html` в браузере. Сборка не требуется.
